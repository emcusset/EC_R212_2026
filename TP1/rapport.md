Rapport — TP1
Points clés
1. Grid et responsive automatique
J’ai appris à utiliser Grid avec auto-fit et minmax pour créer une grille responsive sans media queries complexes. Le navigateur adapte automatiquement le nombre de colonnes selon l’espace disponible.
2. Flexbox pour l’alignement
J’ai utilisé Flexbox pour organiser le header et le footer. Avec justify-content: space-between, j’ai pu aligner facilement le logo à gauche et les liens à droite sans utiliser de marges fixes.
3. Variables CSS pour la maintenance
L’utilisation de variables CSS dans :root permet de centraliser les couleurs et espacements. Cela rend le code plus maintenable et permet de modifier rapidement le design global.
Utilisation de l'IA
Démarche
J’ai utilisé ChatGPT pour m’aider à structurer mon CSS, comprendre comment combiner Grid et Flexbox et respecter les contraintes de simplicité du TP.
Apports et limites
L’IA m’a aidé à améliorer la structure du code et à simplifier certaines parties (notamment avec l’utilisation de gap). Elle m’a aussi aidé à comprendre certains bugs. En revanche, certaines propositions n’étaient pas adaptées (erreurs de syntaxe ou solutions trop complexes), j’ai donc dû vérifier et corriger.
Prompts significatifs
"comment faire une grille responsive simple avec grid" → m’a aidé à comprendre auto-fit et minmax
"comment aligner un logo à gauche et un menu à droite en CSS" → m’a aidé avec Flexbox
